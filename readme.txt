1. gesture-analysis-code.R contains R script for Dresang, Wong, & Buxbaum (2023) behavioral analysis.
2. gesture-items.txt includes pantomimed gestures used for this experiment (Buxbaum et al., 2014 for additional details).
3. gesture-procedure.txt explains details of the gesture imitation experiment task and coding protocol.
4. neuroimaging-procedure.txt explains details of the neuroimaging data acquisition and analysis protocol.
Please contact the authors for additional information.
